## use hexo start my blog

### the steps
1. tools
    * node
    * npm
    * git
    * windows7
2. 依赖
    * 如果需要先修改镜像
    * npm config ls
    * npm config set registry http://registry.npm.taobao.org/
```
npm install -g hexo-cli
npm install -g hexo-server
hexo init
hexo n "comment"
hexo g
hexo s
hexo d

```
3. later step
```
hexo n ""
hexo clean
hexo g
hexo d
hexo s
```
