### 基本步骤

* 两步启动，选网络
* 有四栏
  * * 1：抓取的数据包栏
  * 2：单个数据包的具体信息
  * 3: 字节栏，16进制
  * 4: 状态栏
* 使用过滤器
  ```
  1：只显示与本机相关操作

  ip.addr==192.168.100.10

  2:目的地是我的主机

  ip.dst==192.168.100.10

  3:协议过滤

  http

  4:查看百度服务器主机ip

  ping www.baidu.com

  5:与百度交互

  ip.addr==192.168.100.1002&&ip.dst==39.156.66.14

  6：捕获过滤器

  ipv4 addr==192.168.10.1020

  7；ftp.uzz.edu.cn

  ```
